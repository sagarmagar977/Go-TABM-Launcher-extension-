go

Go TABM Launcher

Local tab and bookmark launcher
Launcher active
Address-bar commands are enabled.
How to launch
1
Press Alt+D to focus the address bar.
2
Type go, then press Space or Tab.
3
Type a query or mode command, then press Enter.
Modes
query Smart mode: tabs first, bookmarks later.
tab query Search open tabs and switch to one.
bm query Search saved bookmarks and folders.
? Show help suggestions.

Fully local. No analytics, no cloud, and no history search from this extension.
